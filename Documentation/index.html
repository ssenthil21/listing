ListingHub - React Next.js Multipurpose Template

Introduction

ListingHub Next is the best business directory & listings template build on Next.js v15.x, React Js v19.x, Bootstrap v5.x with more features and developer friendly. It is supported on all modern & old browsers, tablets, and mobile devices. ListingHub is a high-quality and well-organized Next.js Template specially designed to fit all the needs of a Local Busines Directory, Business review, Online business directory, City Listing, Restaurants Listing, Entertainments, Car/Motor Listing, Local Listing or any other Business Directory Services.

Javascript & Typescript Versions Available.

Command Description

How to change the fonts family in ListingHub?

Please follow the step by step process

First of all add font family improt link from google fonts in _variables.scss @import url('https://fonts.googleapis.com/css2?family=Manrope:wght@400;500;600&display=swap');

Step 1

Open the layout.js (path: ListingHub_Nextjs/src/app/layout.js) file and please import the font on top of the page import { Manrope } from 'next/font/google'

Step 2

Import your font family in layout.js (path: ListingHub_Nextjs/src/app/layout.js) file, or replace your font name in layout.js (path: ListingHub_Nextjs/src/app/layout.js) file.

import { manrope } from 'next/font/google' - Themes primary font family is manrope
Step 3

Pass your font family in layout.js (path: ListingHub_Nextjs/src/app/layout.js) file, or replace your font name in layout.js (path: ListingHub_Nextjs/src/app/layout.js) file.

const manrope = Manrope({
subsets: ['latin'],
weight:['100','200','300','400','500','600','700','800','900'],
variable: '--font-manrope',
})

- Themes primary font family is Manrope

Step 4

change your font family in _variables.scss(path: ListingHub_Nextjs/src/app/assets/scss/_variables.scss) and for dark theme _variables.scss (path: ListingHub_Nextjs/src/app/assets/scss/dark/_variables.scss) file

$font-family-base:    var(--font-manrope);
$font-family-secondary:    var(--font-manrope);

How to change the Theme primary color in ListingHub?

If you change the theme or template color please follow the below step, Please first of all open the _variables.js (path: src/app/assets/scss/_variables.scss) find the $primary and replace or changes with color code

like:
$primary:   #2a52c3; replace only #2a52c3 to your color (#f89d36)
